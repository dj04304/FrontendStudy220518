제이쿼리 -> 자바스크립트 라이브러리 (자바스크립트: 웹브라우저에 동적인 요소를 주기 위함)
            
            크로스브라우징을 구현하기 위한 일종의 자바스크립트 라이브러리
            -크로스 브라우징이란 호환성, 즉 어떤 웹 사이트이든 똑같이 구현할 수 있게끔 호환할 수 있는 것을 말한다.
            
            모든 브라우저에서 동작하는 클라이언트 사이드 언어
            -클라이언트 사이드 언어란, JAVA C언어 등은 서버언어이지만, Javascript 등은 웹 브라우저에서 해석되는 것을 말한다.

            자바스크립트 호환문제/ 웹상에서의 플래시를 대체하기위해 나옴. 

            제이쿼리는 자바스크립트에 비해서 코드가 적게 입력되었지만, 효과는 최대의 효과를 볼 수 있다. 
                -> 간단하고 코딩이 직관적이다.

            DOM (document object model) 에 관련해서 작업처리가 되고 일관된 이벤트들로 연결하며 시각적효과, Ajax어플리케이션 개발등을
            쉽게할 수 있다.
